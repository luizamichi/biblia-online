# Bíblia Online

## Descrição
Construir um sistema web em PHP para consulta de versículos da Bíblia.

## Objetivos
- Utilizar o padrão **MVC**;
- Utilizar o padrão **DAO**;
- Utilizar um padrão de projeto, como sugestão aderir ao **[Factory](https://refactoring.guru/design-patterns/factory-method)**;
- Utilizar o **[PDO](https://www.php.net/manual/pt_BR/book.pdo.php)** para acesso ao banco de dados aplicando o padrão de projeto **[Singleton](https://refactoring.guru/design-patterns/singleton)**;
- Construir o **[autoload](https://www.php.net/manual/pt_BR/function.spl-autoload-register.php)** de classes;
- Criar o **roteamento** de URLs;
- Aplicar o uso de **[herança](https://www.php.net/manual/pt_BR/language.oop5.inheritance.php)**;
- Utilizar **[Reflection Class](https://www.php.net/manual/pt_BR/class.reflectionclass.php)**.
